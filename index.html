第一章：Python基础

知识点讲解

  • 变量与命名规范：变量名只能包含字母、数字和下划线，且不能以数字开头，应做到见名知意。
  • 基本数据类型：整数（int）、浮点数（float）、字符串（str）和布尔值（bool）。
  • 运算符：包含算术运算符（+, -, *, /, //, %）和赋值运算符。
  • 输入输出：使用 print() 函数在屏幕上输出结果，使用 input() 接收用户的键盘输入。

重难点解析

数据类型转换的陷阱

从 input() 接收到的输入永远是字符串类型。如果需要进行数学运算，必须先使用 int() 或 float() 进行类型转换。

# 错误示范：会导致字符串拼接
age = input("请输入年龄：")
print(age + 1) # TypeError 报错

# 正确做法：显式转换
age = int(input("请输入年龄："))
print("明年你", age + 1, "岁")

第二章：流程控制

知识要点

  • 缩进规则：Python不使用大括号，而是严格通过缩进（通常是4个空格）来区分代码块的层级。
  • 条件判断：使用 if、elif 和 else 根据条件的真假来决定程序的走向。
  • 循环结构：for 循环常用于遍历序列或指定次数；while 循环用于当条件为真时持续执行。
  • 循环控制：break 用于强行终止整个循环；continue 用于跳过当前这一轮，进入下一次循环。

重难点解析

条件判断的逻辑与陷阱

在 if-elif 结构中，程序会从上到下依次判断。一旦某个条件满足，执行完对应的代码块后就会直接跳出整个结构，不会再判断下面的条件。

score = 85
# 推荐的写法：条件从严格到宽松
if score >= 90:
    print("优秀")
elif score >= 80:
    print("良好") # 此处会被执行
else:
    print("继续努力")

第三章：函数与模块

知识要点

  • 定义函数：使用 def 关键字声明一个函数，将重复的代码封装起来。
  • 参数传递：支持必须参数、默认参数（带有默认值）以及不定长参数。
  • 函数返回值：使用 return 将计算结果返回给调用者，如果没有写 return，函数默认返回 None。
  • 模块与包：使用 import 或 from ... import ... 来导入其他文件或 Python 自带的标准库（如 math, random）。

重难点解析

局部变量与全局变量的作用域

在函数内部创建的变量默认是局部变量，只能在函数内部使用。如果需要在函数内部修改全局变量，必须使用 global 关键字进行声明。

count = 0 # 全局变量

def add_count():
    global count # 声明我们要修改全局的 count
    count += 1
    print(f"当前计数：{count}")

第四章：数据结构

知识要点

  • 列表 (List)：最常用的容器，有序且可修改，用中括号 [ ] 表示。可以使用 append() 等方法增加元素。
  • 元组 (Tuple)：有序但不可修改，用小括号 ( ) 表示，通常用于存储安全的数据。
  • 字典 (Dictionary)：无序的键值对（Key-Value）集合，用大括号 { } 表示。通过键来快速查找对应的值。
  • 集合 (Set)：无序且不重复的元素集合，同样用大括号表示，常用于去重和关系测试（交集、并集）。

重难点解析

列表的遍历与切片

使用切片 list[start:end:step] 可以非常方便地截取或翻转列表，它是 Python 最强大的特性之一。

nums = [10, 20, 30, 40, 50]

# 截取前三个元素：[10, 20, 30]
first_three = nums[:3]

# 翻转整个列表：[50, 40, 30, 20, 10]
reversed_nums = nums[::-1]
